cnchar.js
获取汉字的拼音和笔画数的js库
View project on github
功能
完整拼音
拼音首字母
笔画数
安装使用
1.script 标签方式引入
引用： <script src="https://www.theajack.com/cnchar/cnchar.min.js"></script>
或点击复制源码
2.npm 方式安装
npm install cnchar
API文档
.spell()
获取汉字完整拼音
如何使用
"测试".spell() 查看
"测试".spell("up") 查看
"测试".spell("low") 查看
'upper','lower'都合法;不区分大小写
非汉字会返回原字符串
